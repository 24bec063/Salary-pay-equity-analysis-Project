# Data Directory

This directory contains datasets used in the Machine Learning project.

## Data Files

### survey_results_public.csv
- **Description**: Stack Overflow Developer Survey 2024 dataset
- **Size**: ~153 MB
- **Columns**: Contains salary, experience, education, country, and other developer-related information
- **Purpose**: Used for salary prediction and fairness classification models

## Notes

- Original data files are tracked with `.gitignore` (files are too large for Git)
- Download instructions and data source links should be documented in the main README.md
